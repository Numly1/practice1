# AutoParts — простой сайт автозапчастей

В архиве готовый статический сайт:

- `index.html` — главная страница
- `catalog.html` — каталог с поиском и фильтром
- `product.html` — карточка товара
- `cart.html` — корзина
- `contacts.html` — контакты и форма заявки
- `css/style.css` — стили
- `js/main.js` — товары, фильтр, корзина и форма

## Как открыть

1. Распакуйте архив.
2. Откройте файл `index.html` в браузере.
3. Для редактирования откройте папку в VS Code.

Корзина работает через `localStorage`, то есть без сервера.
